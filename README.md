# ![Microchip](https://avatars.githubusercontent.com/u/183646887?s=48&v=4) Peripheral Touch Controller - [ptc]
Enhance your product’s user interface by replacing mechanical push buttons with a touch- or gesture-controlled interface. This is an excellent way to add value and advanced features to your application, increase customer appeal and improve your product’s performance and reliability.
